How to use Course Viewer

Video Player

  • Click any lecture in the left panel to load it
  • Subtitles load automatically — toggle with the CC button
  • Use the subtitle search to jump to any spoken word
  • Your playback position is saved automatically

Course Reader (this panel)

  • Open PDFs, HTML pages, or Markdown files as tabs
  • Paste any URL in the input bar and press +
  • Zoom in/out with the − / + controls

Notes

  • Markdown editor with live preview — click the Preview tab
  • Notes persist per directory in localStorage
  • Export notes to a .md file at any time

Run locally for full features

The local version adds video streaming, subtitle sync, directory browsing, and session management.

Windows

powershell -c "iwr https://raw.githubusercontent.com/patchamama/Course-viewer/main/start.bat -OutFile start.bat"; .\start.bat

macOS / Linux

curl -fsSL https://raw.githubusercontent.com/patchamama/Course-viewer/main/start.sh -o start.sh && bash start.sh
